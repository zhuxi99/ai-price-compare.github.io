输入站点，一次抓取模型与分组倍率

兼容 New API、One API、Veloera，以及 One Hub / Done Hub 常见接口。抓取完成后会自动识别模型家族、计算人民币价格，并合并到最新价格 JSON。

连接目标站点

填首页、登录页地址都可以，工具会自动取站点域名。

✓令牌只从浏览器发送给本机 127.0.0.1，再由本机直连你填写的站点；不会写入 JSON、日志、GitHub 或线上网站。
2

选择分组并检查模型

模型 自动分类 模型倍率 输出倍率 分组倍率 输入价 输出价
没有符合筛选条件的模型

价格按 New API 规则换算：倍率 1 对应 2 美元 / 1M tokens，再乘分组倍率和美元汇率。接口未提供缓存倍率时，缓存输入价暂按普通输入价计算；按次计费模型不会写入当前价格表。

3

生成新的发布数据

保存时会读取“下载”目录中最新的有效价格 JSON：同一站点、同一模型自动更新，其余记录保持不变；新模型自动加入。
